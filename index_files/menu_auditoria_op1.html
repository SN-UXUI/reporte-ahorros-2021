Consideraciones:
La base muestral son las cotizaciones adjudicadas a un proveedor, que pueden tener o no una orden de compra.
El monto ahorrado es la diferencia entre, el promedio ofertas unitario recibidas menos el precio adjudicado unitario, multiplicado por la cantidad adjudicada.
El % ahorro es 1 menos el cociente entre el precio adjudicado unitario y el promedio oferta unitario.
Se consideran valores netos de IVA.
Con el objetivo de disminuir el impacto de aquellas ofertas erróneas o mal ingresadas, el cálculo del ahorro contenido en el presente informe tiene incorporado la desviación estándar.
